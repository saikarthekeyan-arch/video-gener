# AI Video Generator

Generate real AI-powered videos in your browser - 100% free, no paid tiers.

## Features

- **AI Video Generation** - Create videos from text prompts using Stable Diffusion
- **8 Art Styles** - Cinematic, Anime, Realistic, Fantasy, 3D, Watercolor, Cyberpunk, Oil Painting
- **Video Tools** - Slideshow, Merge, Convert formats
- **All Free** - Uses Hugging Face free inference API

## How to Use

1. Get a **FREE** Hugging Face token at https://huggingface.co/settings/tokens
2. Paste token in the app
3. Describe your video prompt
4. Choose style and settings
5. Click Generate AI Video!

## Deploy to Vercel (Free)

1. Push files to GitHub
2. Go to https://vercel.com
3. Import your repository
4. Deploy!

Or use Netlify, Cloudflare Pages, GitHub Pages.

## Tech Stack

- Hugging Face Inference API (Free)
- FFmpeg.wasm 0.11.x (Browser-based)
- Stable Diffusion models
- Pure HTML/CSS/JavaScript
